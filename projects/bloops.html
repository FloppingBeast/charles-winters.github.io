Bloops

Gameplay Systems Programmer — Global Game Jam — 48 Hours

Overview

Bloops was a 48-hour Global Game Jam project built by a nine-person team in Unity. The game is a top-down puzzle experience where players redirect bubbles using interactive bathroom objects. I focused primarily on core gameplay systems and bubble movement mechanics.

Contributions

Sponge Bounce

Implemented a directional sponge mechanic that redirects bubbles at a 90° angle while blending movement along the sponge surface.

Fan Interaction

Built a raycast-based airflow mechanic that dynamically changes bubble velocity based on fan orientation.

Parabolic Launcher

Created parabolic arc trajectories that integrate cleanly back into the grid.

Bubble Movement

Implemented grid-aligned velocity movement with recentering logic to keep bubbles stable and visually smooth.

Outcome

The team delivered a polished, playable build within 48 hours and successfully submitted it to this Global Game Jam webpage. The project demonstrated strong rapid prototyping, cross-discipline collaboration, and my ability to quickly adapt to new tools while maintaining technical quality under pressure.
