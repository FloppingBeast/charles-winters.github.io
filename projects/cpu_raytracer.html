CPU Ray Tracer

Overview

The purpose of this project was to build a deeper understanding of ray tracing by implementing a CPU-based ray tracer from scratch. The work explores core techniques such as ray–geometry intersection testing, illumination and shading models, and antialiasing. In addition to the mathematical and physical principles of ray tracing, this project emphasizes practical implementation and core design decisions involved in constructing a functional renderer.
